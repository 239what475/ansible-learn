# 仓库约定

## 项目目标
- 这是一个面向初学者的 `SRE tools` 学习仓库。
- 当前最完整的学习主线位于 `ansible/` 子目录。
- 后续可能继续扩展 `kubernetes/`、`terraform/` 等工具主线。

## 目录约定
- 根目录优先保留总导航和总说明。
- 各工具的具体学习内容放在各自子目录下，例如：
  - `ansible/`
  - 未来的 `kubernetes/`
  - 未来的 `terraform/`
- 如果某个子目录有更具体的约定，应在该子目录下新增自己的 `AGENTS.md`。

## 文件约定
- 需要用户复制后再按本机环境修改的文件，一律以 `.example` 结尾提交到仓库。
- 用户本地实际使用的文件去掉 `.example` 后缀，并加入对应目录的 `.gitignore`。

## Git 忽略约定
- 优先在对应子目录放置局部 `.gitignore`，避免根目录 `.gitignore` 过大。
- 根目录 `.gitignore` 只保留全局通用的忽略项。

## 编写原则
- 保持改动小而集中。
- 优先保证教程清晰、易懂、可操作。
- 默认使用中文编写教程和说明，除非用户明确要求其他语言。
- 未经用户确认，不主动扩展当前学习步骤之外的内容。
