# SRE Tools Learn

这是一个面向初学者、按工具逐步扩展的 `SRE` 学习仓库。

当前已经完整整理好的学习主线是：

- `ansible/`：从 `00` 到 `30` 的 `Ansible` 教程与 mini project

后续计划继续补充：

- `kubernetes/`
- `terraform/`
- 以及更多常见的 `SRE / Platform Engineering` 工具链内容

## 当前推荐入口

如果你现在要继续学习，请先从这里开始：

- `ansible/README.md`
- `ansible/ROADMAP.md`
- `terraform/README.md`
- `terraform/ROADMAP.md`
